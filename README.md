# QR Code Generator

Projeto desenvolvido com o objetivo de gerar QR Codes utilizando Java.

## Status do projeto

⚠️ **Desenvolvimento interrompido temporariamente.**

O projeto foi pausado devido à necessidade de integração com serviços da AWS. No momento, não foi possível concluir essa etapa por falta de acesso a uma conta AWS ativa com as credenciais necessárias (`Access Key` e `Secret Key`).

O desenvolvimento poderá ser retomado futuramente, quando houver acesso aos recursos necessários.

## Tecnologias utilizadas

* Java
* Maven
* Git
* GitHub

## Objetivo

Este projeto faz parte dos meus estudos em desenvolvimento Back-end com Java e tem como objetivo praticar a criação de aplicações, organização de código e integração com serviços externos.
